# QonText

QonText is a simple web application that helps teams, founders, students, and developers create professional assignment and project briefs in Markdown format.

Users can enter task details such as the project name, assignee, description, requirements, deliverables, and deadlines. The application automatically generates a clean, well-structured README file that can be previewed, copied, or downloaded as a `.md` file.

The goal is to make task delegation and project documentation faster, more consistent, and easier to share across teams, especially in environments where GitHub, Markdown readers, and documentation-first workflows are used.

## Features

* Create structured task and assignment briefs
* Generate Markdown (`README.md`) files automatically
* Live Markdown preview
* Download generated files as `.md`
* Clean and simple user interface
* Suitable for software projects, academic work, startup teams, and freelance collaboration

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* React Markdown
